Property Onboarding — How Landlords Add Properties

Landlords add properties through a guided “Add Property Wizard”.

Think of it like the listing flow used by platforms such as Airbnb or Zillow — simple, step-by-step.

Step 1 — Click “Add Property”

From the landlord dashboard:

Button:

+ Add Property

This opens the Property Creation Wizard.

Step 2 — Enter Property Address

Fields required:

Street address
City
Province
Postal code
Country

Features:

Google Maps auto-fill

Address validation

Detects jurisdiction automatically

Example logic:

Address entered → Toronto → Ontario lease template loaded

Why this matters:

The system automatically prepares correct legal rules for that location.

Step 3 — Select Property Type

Options:

Apartment building
Single-family home
Duplex / Triplex
Condo
Student housing
Commercial property
Mixed-use property

This helps the system customize lease templates and analytics.

Step 4 — Enter Number of Units

Example:

How many units does this property have?

Input:

1
2–5
6–20
20+

If more than one unit:

System creates unit manager dashboard.

Step 5 — Add Units

Each unit requires:

Unit number
Bedrooms
Bathrooms
Square footage
Rent price
Availability date
Furnished or unfurnished
Parking
Pets allowed

Example:

Unit 201
2 bedroom
1 bathroom
$1950/month
Available May 1
Step 6 — Upload Photos

Landlord uploads:

Unit photos
Building photos
Amenities photos

Optional but strongly recommended.

AI feature:

AI checks image quality
AI suggests missing photos
Step 7 — Amenities & Features

Checklist:

Laundry
Parking
Air conditioning
Heating
Balcony
Gym
Pool
Elevator
Security

This helps the AI pricing engine.

Step 8 — Set Rent & Lease Terms

Fields:

Monthly rent
Security deposit
Utilities included
Lease length
Minimum income requirement

AI suggestion example:

Market rent in this area: $2100
Your price: $1950

Suggested increase: +$100

Landlord can accept or ignore.

Step 9 — Tenant Requirements

Optional but helpful filters.

Examples:

Minimum income
Guarantor allowed
Students allowed
Pets allowed
Smoking allowed

This improves screening accuracy.

Step 10 — Publish Listing

Buttons:

Save as Draft
Publish Listing

When published:

The unit appears in:

Tenant search
Application system
Landlord dashboard

Optional distribution:

Facebook Marketplace
Rental websites
Student housing boards
What Happens After Property Is Added

Once a property is added:

System automatically creates:

Property dashboard
Unit management page
Application inbox
Payment ledger
Maintenance tracking

This means the landlord can immediately:

accept applications
generate leases
collect rent
manage tenants
Safety Checks When Adding Property

To prevent fraud:

Verification includes:

Address validation
Owner verification
Duplicate property detection

Example:

Two users adding same property → flagged for review
Simplified UX Version

For small landlords (1–2 units), the system can offer Quick Add Mode:

Only asks:

Address
Unit type
Rent
Availability

Everything else can be edited later.
