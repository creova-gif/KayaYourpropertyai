6. Design System Parameters

Colors: White / Black base with accent colors for CTA & alerts

Fonts: Modern sans-serif, readable, hierarchy for headings vs body

Spacing: Consistent grid, 8–16px increments for mobile, 24–32px for desktop

Components: Buttons, cards, tables, modals, toast notifications, progress bars

Branding: Bold clean typography, trust indicators (verified badges, padlocks for security)

7. Property & Application Parameters

For each property:

Address, city, province, postal code

Property type (apartment, house, commercial)

Units: bedrooms, bathrooms, square footage

Amenities: parking, laundry, pets

Lease terms: rent, deposit, utilities included

Tenant requirements: minimum income, guarantor, allowed pets

For each tenant application:

Personal info: name, email, phone

Documents: ID, income proof, references

AI scoring: risk, income/rent ratio, completeness

Status tracking: pending → approved → lease signed → active
